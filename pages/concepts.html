Hierarquia de títulos

Os títulos das páginas possuem tags com uma hierarquia simples de níveis. Os níveis vão de 1 até 6:
  • <h1;>
  • <h2;>
  • <h3;>
  • <h4;>
  • <h5;>
  • <h6;>

Formatação de texto

Existem diversas tags para formatação, inclusive tags que foram obsoletas no HTML5 devido a não se encaixar num significado semântico, como a tag <big> e < small>. A tag <small> ainda é considerada por ter um sentido, como as letras pequenas de um documento; já a tag <big> não consegue ter um outro sentido além de forma.
Algumas tags conhecidas:
  • <strong> - Negrito
  • <em> - Itálico
  • <del> - Riscado
  • <ins> - Sublinhado
  • <mark> - Marcação
  • <sup> - Sobrescrito
  • <sub> - Subscrito
  • <code> - Códigos
  • <pre> - Formatação para códigos;
  • <q> - Citação
  • <blockquote> - Citação em blocos
  • <abbr> - Abreviações
  • <bdo> - bi-directional override
  • <rtl> - Da direita pra esquerda
  • <ltr> - Da esquerda pra direita
