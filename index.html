Para relembrar:

HTML não trabalha com formas, somente semântica. Tudo que envolve forma é CSS. Existe uma maneira extremamente básica que afeta somente a tag com o parâmetro, que é CSS dentro de uma tag. Por exemplo, utilizar a tag <mark> e trocar a cor. Utilizando o parâmetro style, podemos inserir uma CSS simples dentro de uma tag, mas afeta somente a tag que o estilo foi inserido. Símbolos são inseridos utilizando o & com prefíxos de seus nomes. como ™, a famosa trade mark, e ©, o símbolo de copyright. Alguns outros exemplos são ¥, $, ↑, ↓...
